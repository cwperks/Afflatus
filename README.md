# Afflatus

## Requires

- Rails (using 5 locally, but >= 4 should work)
- Ruby (using 2.3.3 locally, but any should be fine)

## Install and Run

- `git clone git@github.com:cwperks/Afflatus.git`
- `cd Afflatus`
- `bundle install`
- `rails s -p 3000`
- Navigate to browser and go to `127.0.0.1:3000`
